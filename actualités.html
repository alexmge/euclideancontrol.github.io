Notes de patch

Patch 0.1

Dans la dernière mise à jour, nous avons ajouté les animations de quelques mouvements indispensables de notre jeu comme par exemple le tir, le saut, la marche... Vous pouvez voir ci-dessous l'animation du tir du personnage avec l'arme principale. Cependant, nous avons eu quelques difficultés lors du développement surtout pour adapter tout ce que nous faisons aux portails et au code qu'ils utilisent. Nous avons également eu quelques problèmes avec le multijoueur que nous rencontrons encore actuellement. Dans la globalité, les problèmes rencontrés ne sont pas si dérangeants qu'ils n'y paraissent
